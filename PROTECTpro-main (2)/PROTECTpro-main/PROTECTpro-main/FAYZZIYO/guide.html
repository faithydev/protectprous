02 URL Skanerlash

Tez Skanerlash (Quick Scan)

1-3 soniya ichida asosiy xavflarni aniqlaydi:

Quick Scan Process
$ protectpro scan --quick https://example.com
✓ Checking domain reputation...
✓ Analyzing URL structure...
✓ Checking typosquatting...
✓ WHOIS analysis...
RESULT: SAFE (Risk Score: 0/100)
Tekshiriladigan parametrlar:
  • Domain reputatsiyasi (Google Safe Browsing, VirusTotal)
  • Typosquatting (google.com → gooogle.com)
  • URL tuzilmasi (base64, eval, obfuscation)
  • WHOIS ma'lumotlari (domen yoshi, registrator)

Chuqur Tahlil (Deep Scan)

20-60 soniya ichida to'liq sandbox tahlili:

Deep Scan Stages 
Phase 1: Headless Browser Launch (5s)
  → Chromium sandbox environment
  → JavaScript isolation enabled

Phase 2: Page Load & Network Monitor (15-30s)
  → Capturing all HTTP/HTTPS requests
  → WebSocket connections tracking
  → sendBeacon() calls detection

Phase 3: Behavior Analysis (10-20s)
  → eval() and obfuscated code detection
  → Data exfiltration attempts
  → Cookie/localStorage access patterns
  → Suspicious redirects

Phase 4: Report Generation (5s)
  → Risk score calculation
  → Detailed threat breakdown
  → Screenshot & HAR export

03 API Integratsiya

API Kalitini Olish

Dashboard → Settings → API Keys bo'limidan API kalitingizni yarating.

API Endpoint:

Misol: Quick Scan

JavaScript 
const API_KEY = 'your_api_key_here';
const URL_TO_SCAN = 'https://example.com';

async function quickScan(url) {
  const response = await fetch('https://api.protectpro.uz/v1/scan', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${API_KEY}`
    },
    body: JSON.stringify({
      url: url,
      scan_type: 'quick'
    })
  });
  
  const result = await response.json();
  console.log('Scan Result:', result);
  return result;
}

quickScan(URL_TO_SCAN);
