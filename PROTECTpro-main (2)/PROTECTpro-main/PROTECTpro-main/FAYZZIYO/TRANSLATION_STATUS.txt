IMPLEMENTATION DETAILS:
=================================================================

Technology Stack:
  • Custom i18n System: data-i18n attributes on HTML elements
  • Language Storage: localStorage for persistent language selection
  • Dynamic Translation: JavaScript updates all text on language switch
  • Centralized Translations: Single translations.js file (1225 lines)

How It Works:
  1. HTML elements marked with data-i18n="section.key" attribute
  2. User selects language via buttons (UZ/EN/RU)
  3. JavaScript fetches translation from translations object
  4. All text content updates dynamically
  5. Language preference saved in localStorage
